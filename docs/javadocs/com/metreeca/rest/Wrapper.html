Interface Wrapper

  • All Known Subinterfaces:
    Engine
    All Known Implementing Classes:
    Aliaser, Bearer, Controller, CORS, Driver, Gateway, GraphEngine, Validator
    Functional Interface:
    This is a functional interface and can therefore be used as the assignment target for a lambda expression or method reference. 

    @FunctionalInterface
public interface Wrapper
    Handler wrapper {thread-safe}.

    Inspects and possibly alters requests and responses processed and generated by resource handlers.

    Warning / Implementations must be thread-safe.

        Creates a conditional wrapper.
        Parameters:
        test - the request predicate used to decide if requests and responses are to be routed through the wrapper
        pass - the wrapper requests and responses are to be routed through when test evaluates to true on the request
        Returns:

        Creates a conditional wrapper.
        Parameters:
        test - the request predicate used to select the wrapper requests and responses are to be routed through
        pass - the wrapper requests and responses are to be routed through when test evaluates to true on the request
        fail - the wrapper requests and responses are to be routed through when test evaluates to false on the request
