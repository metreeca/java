    • get

      public <T> T get (Supplier<T> factory)
      Retrieves the shared service created by a factory.

      The new service is cached so that further calls for the same factory are idempotent.

      During object construction, nested shared service dependencies may be retrieved from this toolbox through the static service locator method of the toolbox class. The toolbox used by the service locator method is managed through a ThreadLocal variable, so it won't be available to object constructors executed on a different thread.

      Type Parameters:
      T - the type of the shared service created by factory
      Parameters:
      factory - the factory responsible for creating the required shared service; must return a non-null and thread-safe object
      Returns:
      the shared service created by factory or by its plugin replacement if one was specified
      Throws:
      IllegalArgumentException - if factory is null
      IllegalStateException - if a circular service dependency is detected

      Returns:
      this toolbox

    • exec

      public Toolbox exec (Runnable... tasks)
      Executes a set of tasks using shared services managed by this toolbox.

      During task execution, shared service may be retrieved from this toolbox through the static service locator method of the toolbox class. The toolbox used by the service locator method is managed through a ThreadLocal variable, so it won't be available to methods executed on a different thread.

      Parameters:
      tasks - the tasks to be executed
      Returns:
      this toolbox
      Throws:
      NullPointerException - if task is null or contains null items
