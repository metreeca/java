    • retrieve

      <T> T retrieve (String key, Function<InputStream,T> decoder, Function<OutputStream,T> encoder)
      Retrieves an item from this cache.
      Type Parameters:
      T - the type of the cached items
      Parameters:
      key - the key of the item to be retrieved
      decoder - a function decoding a cached item from its binary representation; takes as argument an input stream for reading the binary representation of the item to be retrieved and is expected to return a non null object of type T
      encoder - a function encoding an item to be cached to its binary representation; takes as argument an output stream for writing the binary representation of the item to be cached and is expected to return a non null object of type T
      Returns:
      an object of type T, returned either from the decoder, if a binary blob matching key was found in the cache, or by the encoder, otherwise
      Throws:
      NullPointerException - if any argument is null
