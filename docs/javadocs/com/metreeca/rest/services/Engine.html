    • create

      Optional<Frame> create (Frame frame, Shape shape)
      Handles creation requests.
      Parameters:
      frame - a frame describing the linked data resource to be created
      shape - a shape describing the frame linked data model
      Returns:
      an optional containing frame, possibly extended with server managed properties, if the frame focus IRI was not already present in the storage backend; an empty optional, otherwise
      Throws:
      NullPointerException - if either frame or shape is null
      Implementation Specification:
      Concrete implementations must assume that frame was already configured with a unique focus IRI for the resource to be created and its model rewritten accordingly, for instance by an outer Creator handler

    • relate

      Optional<Frame> relate (Frame frame, Query query)
      Handles retrieval requests.
      Parameters:
      frame - a frame focused on the linked data resource to be retrieved
      query - a query describing the expected response model
      Returns:
      an optional containing a frame describing the frame focus, possibly filtered according to query, if the storage backend contained a matching linked data resource; an empty optional, otherwise
      Throws:
      NullPointerException - if either item or query is null

    • update

    • delete

      Optional<Frame> delete (Frame frame, Shape shape)
      Handles deletion requests.
      Parameters:
      frame - a frame focused on the linked data resource to be deleted
      shape - a shape describing the frame model to be deleted
      Returns:
      an optional containing a frame describing the deleted frame, if it was present in the storage backend; an empty optional, otherwise
      Throws:
      NullPointerException - if either frame or shape is null
      Implementation Specification:
      If the deleted model is not readily available, concrete implementations may return a frame with an empty model
