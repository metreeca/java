    • encode

      public <M extends Message<M>> M encode (M message, Frame value)
      Configures message Content-Type header to "application/json", unless already defined, and encodes the JSON-LD model value into the output stream accepted by the message OutputFormat body.

      If the originating message request includes an Accept-Language header, a suitably localized version of the message shape is used in the conversion process and only matching tagged literals from value are included in the response body.

      Warning / @context objects generated from the message shape attribute are embedded only if Content-Type is "application/ld+json".

      Overrides:
      encode in class Format<Frame>
      Type Parameters:
      M - the type of message
      Parameters:
      message - the message whose body is to be encoded
      value - the body being encoded into message
      Returns:
      the target message with the encoded value as body
