public final class Creator extends Delegator
Model-driven resource creator.

Handles creation requests on the linked data collection identified by the request focus item:

  • redacts the shape associated with the request according to the request user roles;
  • performs shape-based authorization, considering the subset of the request shape enabled by the Guard.Create task and the Guard.Detail view;
  • validates the JSON-LD request body against the request shape; malformed or invalid payloads are reported respectively with a 400 or a 422 status code;
  • generates a unique IRI for the resource to be created on the basis on the stem of the the request IRI and the value of the Slug request header, if one is found, or a random id, otherwise;
  • rewrites the request body to the assigned IRI and stores it with the assistance of the shared linked data engine; the target collection identified by the request focus item is connected to the newly created resource according to the filtering constraints in the request shape.

On successful completion, generates a response including:

  • a 201 status code;
  • a Location HTTP response header advertising the IRI of the newly created resource.
