Interface Handler

All Known Implementing Classes:
Creator, Delegator, Deleter, Endpoint, Graphs, MessageException, Packer, Publisher, Relator, Router, SPARQL, Updater
Functional Interface:
This is a functional interface and can therefore be used as the assignment target for a lambda expression or method reference.
@FunctionalInterface public interface Handler
Resource handler {thread-safe}.

Exposes and manages the state of a linked data resource, generating responses in reaction to requests.

Warning / Implementations must be thread-safe.
