Interface Engine

  • All Superinterfaces:
    Wrapper
    All Known Implementing Classes:
    GraphEngine 

    public interface Engine
extends Wrapper
    Model-driven storage engine.

    Handles model-driven CRUD operations on resource managed by a specific storage backend.

    When acting as a wrapper, ensures that requests are handled within a storage transaction:

    • if a transaction is not already active on the underlying storage, begins one and commits it on successful handler completion;
    • if the handler throws an exception, rolls back the transaction and rethrows the exception;
    • in either case, no action is taken if the transaction was already terminated inside the handler.

    Falls back to plain handler execution if transactions are not supported by this engine.
