com.metreeca.rest.assets

Interface Engine

  • All Superinterfaces:
    Wrapper
    All Known Implementing Classes:
    GraphEngine 

    public interface Engine
extends Wrapper
    Model-driven storage engine.

    Handles model-driven CRUD operations on resource managed by a specific storage backend.

    When acting as a wrapper, ensures that requests are handled on a single connection to the storage backend.

      • create

        Future<Response> create(Request request)
        Handles creation requests.

        Handles creation requests of the linked data resource identified by the request item possibly using an engine-specific request payload and the message shape.

        Parameters:
        request - a creation request for the managed linked data resource
        Returns:
        a lazy response generated for the managed linked data resource in reaction to the creation request
        Throws:
        NullPointerException - if request is null
        Implementation Specification:
        Concrete implementations must assume that Request.path() was already configured with a unique identifier for the resource to be created and the JSON-LD payload of the request rewritten accordingly, for instance by an outer Creator handler
