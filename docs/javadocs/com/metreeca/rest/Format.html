    • decode

      public Either<MessageException,V> decode (Message<?> message)
      Decodes a message body.

      The default implementation returns a no-op message exception.

      Concrete subclasses should report decoding issues using the following HTTP status codes:

      • Response.UnsupportedMediaType for missing bodies;
      • Response.BadRequest for malformed bodies, unless a more specific status code is available.
      Parameters:
      message - the message whose body is to be decoded
      Returns:
      either a message exception reporting a decoding issue or the decoded message body
      Throws:
      NullPointerException - if message is null
