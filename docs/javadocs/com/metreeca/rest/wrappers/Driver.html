Class Driver

com.metreeca.rest.wrappers.Driver
All Implemented Interfaces:
Wrapper
public final class Driver extends Object implements Wrapper
Shape-based content driver.

Drives the lifecycle of linked data resources managed by wrapped handlers associating a shape to incoming requests

Wrapped handlers are responsible for:

  • redacting the shape associated to incoming request according to the task to be performed;
  • associating a shape to outgoing responses in order to drive further processing (e.g. JSON body mapping).

Warning / Both operations must be performed taking into account the roles of the current request user: no user-related shape redaction is performed by the driver wrapper on behalf of nested handlers.
