      @SafeVarargs public static Validator validator (Function<Request,Collection<String>>... rules)
      Creates a validating preprocessor.

      Validation rules handle a target request and must return a non-null but possibly empty collection of validation issues; if the collection is not empty, the request fails with a Response.UnprocessableEntity status code; otherwise, the request is routed to the wrapped handler.

      Parameters:
      rules - the custom validation rules to be applied to incoming requests
      Returns:
      a new validator
      Throws:
      NullPointerException - if rules is null or contains null values
