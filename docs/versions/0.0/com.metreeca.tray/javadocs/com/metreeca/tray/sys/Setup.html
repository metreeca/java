      • SetupProperty

        public static final String SetupProperty
        Setup property for the path of the custom setup properties file ("setup.properties"={path} || ./"setup.properties").

        This property is intended to be modified through a java command-line flag as -D"setup.properties"=<path>.

        Relative paths paths are interpreted with respect to the current working directory of the process as read from the user.dir system property.

        See Also:
        Constant Field Values
