    Task

    public final class Item
extends Object
implements Task
    Item injection task.

    Generates a stream of parametrized cells.

    For each cell in the feed:

    • computes item parameters evaluating expressions on the cell model;
    • parametrize IRI and text on the computed parameters; if IRI is empty, a new unique IRI is generated;
    • if not empty, uploads the parametrized text to the cache, identifying it with the parametrized IRI;
    • generates a new cell focused on the parametrized IRI; downstream tasks may read the uploaded content from the cache using the focus IRIs of the generated cells.
